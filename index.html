Vem jag är

Jag är en konsult med erfarenhet inom webbutveckling. Jag erbjuder skräddarsydda lösningar för både privatpersoner och företag, med fokus på ASP.NET Core, Blazor och modern front-end utveckling.

Teknisk kompetens

Jag arbetar med moderna teknologier som ASP.NET Core, C#, Blazor, React, HTML5, CSS3 och JavaScript. Jag hjälper företag att skapa webbapplikationer med fokus på funktionalitet och användarupplevelse.

  • 🚀 C# & JavaScript utveckling
  • ⚛️ Blazor & React applikationer
  • 🎨 Responsiv design & UX/UI

Tjänster

Lösningar inom ASP.NET Core, Blazor och modern front-end utveckling

ASP.NET Core

Jag utvecklar webbapplikationer med C# och ASP.NET Core. Fokus ligger på att bygga funktionella och skalbara backend-lösningar med modern arkitektur.

  • ASP.NET Core Web API
  • MVC applikationer
  • Database integration
  • Authentication & Authorization

Blazor Applikationer

Jag utvecklar klient-sidiga applikationer med Blazor. Genom att använda C# för både front-end och back-end blir koden mer konsistent och lättare att underhålla.

  • Blazor Server & WebAssembly
  • Komponentbaserad utveckling
  • Real-time funktionalitet
  • PWA integration

Modern Front-end

Jag skapar moderna och responsiva gränssnitt med fokus på kodkvalitet och prestanda. Med HTML5, CSS3 och JavaScript bygger jag lösningar som fungerar bra på alla plattformar.

  • Responsiv design
  • CSS Grid & Flexbox
  • JavaScript ES6+
  • React integration
